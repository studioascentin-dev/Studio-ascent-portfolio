# **App Name**: Studio Ascent

## Core Features:

- 3D Welcome Sequence: Interactive 3D welcome area: Greet visitors with a personalized, animated 3D scene showcasing key skills and featuring the name 'Dev Kumar Das'.
- 3D Project Display: Display project samples (video editing, photo editing, PPT, web design, web development) within interactive 3D environments or models. Use Nextjs to allow seamless embedding of videos.
- Category Selection: Allow users to explore different service categories (video editing, photo editing, etc.) through interactive 3D elements, ensuring a visually engaging and intuitive browsing experience.
- Pricing Reveal: Display pricing for each service category in a dynamic, visually appealing 3D layout, making it easy for potential clients to understand costs.
- Animated 3D Buttons: Create eye-catching, animated 3D buttons for key actions, encouraging user interaction.
- Smooth Scroll Animation: Implement seamless scrolling effects with each section appearing smoothly, enhancing the overall user experience. Use Nextjs to trigger animations upon screen position changes.
- Intelligent Project Selection: Integrate a tool to suggest the most relevant project examples to a potential client based on their initial selections in the interactive welcome area. This helps tailor the experience and highlights the most impactful work.

## Style Guidelines:

- Primary color: Deep rust (#A84444) for a vibrant and energetic feel.
- Background color: Light tan (#F5E7D2). This provides a warm and inviting backdrop with good contrast.
- Accent color: Forest green (#228B22). This adds a touch of sophistication and modernity.
- Font pairing: 'Oswald' (sans-serif) for headers and short blocks of text, and 'Roboto' (sans-serif) for longer blocks of body text.
- Modern, minimalist 3D icons to represent each service category.
- Full-screen layout with interactive 3D elements layered strategically.
- Subtle animations and transitions for a fluid, immersive experience.
- Use gradient color for buttons (mix color of two or three).
